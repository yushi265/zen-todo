# Commands

ZenTodo registers the following commands in the Obsidian command palette (prefix: `ZenTodo:`):

| Command | Description |
|---------|-------------|
| `ZenTodo: Open todo list` | Opens the ZenTodo sidebar panel. Creates one if not already open. |
| `ZenTodo: Create new todo list` | Opens the panel and immediately triggers the new list creation flow. |

## Opening ZenTodo

In addition to the command palette, you can open ZenTodo via:

- The **checkbox icon** (☑) in the ribbon (left sidebar)
