# Drag and Drop

ZenTodo supports drag & drop for both task rows and list tabs.

## Reordering Tasks

1. Hover over a task row to reveal the drag handle (⠿) on the left edge.
2. Click and drag the handle to reorder the task.
3. A drop indicator line shows where the task will be placed.
4. Release to drop.

### Constraints

- Tasks can only be reordered **within the same section** (incomplete tasks within incomplete, completed within completed).
- Drag does not move tasks across sections.

### Canceling

Press **Escape** during a drag to cancel without reordering.

## Reordering List Tabs

1. Click and drag any list tab.
2. A vertical indicator line shows the insertion point.
3. Release to drop. The new order is saved automatically.

### Canceling

Press **Escape** during a tab drag to cancel.
