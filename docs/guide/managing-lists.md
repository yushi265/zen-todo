# Managing Lists

## Creating a List

Click **+** in the tab bar. Enter a name and press **Enter**. The list is saved as a `.md` file in your configured todo folder.

## Switching Lists

Click any tab in the tab bar to switch to that list.

## The "All" Tab

The first tab shows tasks from **all lists combined**. Completed tasks are hidden in this view to keep it focused on what needs to be done.

## Reordering Tabs

Drag any tab to reorder it. The order is saved automatically and persists across Obsidian restarts.

## Deleting a List

To delete a list, delete the corresponding `.md` file from your vault. ZenTodo will detect the change and remove the tab.
