# Quick Start

## Opening ZenTodo

After installation, open ZenTodo using any of these methods:

- Click the **checkbox icon** (☑) in the ribbon (left sidebar)
- Command palette: `ZenTodo: Open todo list`
- Command palette: `ZenTodo: Create new todo list`

## Creating Your First List

1. Click **+** in the tab bar to create a new list.
2. Enter a name for your list and press **Enter**.
3. The list file is saved as a `.md` file in your configured todo folder (default: `30_ToDos`).

## Adding Your First Task

1. Click the input field at the top of the panel.
2. Type your task text.
3. Optionally click the calendar icon to set a due date.
4. Press **Enter** or click **Add**.
