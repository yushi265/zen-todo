# 設定

**設定 → ZenTodo** でプラグインを設定できます。

## 設定項目

| 設定 | デフォルト | 説明 |
|------|-----------|------|
| **Todo フォルダ** | `30_ToDos` | リストの `.md` ファイルを保存する Vault フォルダ。Vault ルートからの相対パス。OS に関わらず `/` を使用。 |
| **デフォルトで完了タスクを表示** | オフ | 有効にすると、リストを開いたときに完了タスクセクションが展開されます。 |
| **親タスクの自動完了** | オン | 親タスクの全サブタスクが完了すると自動的に親も完了にします。サブタスクを再度開くと親も再開されます。 |

## Todo フォルダの変更

1. **設定 → ZenTodo** を開く。
2. **Todo フォルダ** のパスを変更。
3. 既存の `.md` ファイルを新しいフォルダに移動。

::: warning
ZenTodo は設定されたフォルダのみを監視します。フォルダ外のファイルは表示・更新されません。
:::
