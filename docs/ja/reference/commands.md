# コマンド

ZenTodo は Obsidian のコマンドパレットに以下のコマンドを登録します（プレフィックス: `ZenTodo:`）：

| コマンド | 説明 |
|---------|------|
| `ZenTodo: Open todo list` | ZenTodo サイドバーパネルを開きます。まだ開いていない場合は作成します。 |
| `ZenTodo: Create new todo list` | パネルを開いて即座に新しいリスト作成フローを開始します。 |

## ZenTodo を開く

コマンドパレット以外にも、以下の方法で開けます：

- リボン（左サイドバー）の **チェックボックスアイコン**（☑）
