# タスク操作

## タスクを追加する

パネル上部の入力フィールドにテキストを入力して **Enter** を押すか **追加** をクリック。

### 期日を設定する

送信前にカレンダーアイコンをクリックすると期日を設定できます。期日は Markdown ファイルに `📅 YYYY-MM-DD` として保存されます。

## タスクアクション

タスクにホバーすると右側にアクションボタンが表示されます：

| アイコン | アクション | 表示条件 |
|---------|-----------|---------|
| ⠿ | ドラッグハンドル — ドラッグして並び替え | 常時 |
| 📅 | 期日を設定 / 変更 | 常時 |
| ➕ | サブタスクを追加 | 未完了のルートタスクのみ |
| 🗃️ | タスクをアーカイブ | 完了済みのルートタスクのみ |
| 🗑️ | タスクを削除 | 常時 |

## タスクを完了する

チェックボックスをクリックしてタスクを完了にします。完了日（`✅ YYYY-MM-DD`）が自動的に追加されます。

## インライン編集

タスクのテキストをクリックするとインラインで編集できます。**Enter** で保存、**Escape** でキャンセル。

## サブタスク

未完了のルートタスクの **➕** アイコンをクリックするとサブタスクを追加できます。サブタスクは Markdown ファイルにタブ文字でインデントされて保存されます。

### 親タスクの自動完了

**親タスクの自動完了** 設定が有効（デフォルト: オン）の場合、全サブタスクが完了すると親タスクも自動的に完了になります。サブタスクを再度開くと親タスクも再開されます。

## 期日の色

期日があるタスクは色で状態を表示します：

- **赤**: 期限超過（期日を過ぎた未完了タスク）
- **オレンジ/アクセント**: 今日が期日
- **通常**: 将来の期日
