# インライン埋め込みビュー

## ノートに ZenTodo を埋め込む

ノートの任意の場所に `zen-todo` フェンスコードブロックを追加すると ZenTodo パネルをインラインで埋め込めます：

````markdown
```zen-todo
```
````

サイドバーパネルと同じリストとアクションが使えるフルインタラクティブな ZenTodo パネルが表示されます。

## クエリモード

コードブロックにフィルタ式を書くと、全リストから絞り込んだタスク結果を表示します：

````markdown
```zen-todo
completed today
```
````

クエリモードでは埋め込みビューから直接タスクの完了を切り替えられます。変更は元の `.md` ファイルに即座に保存されます。

フィルタの完全なリファレンスは [クエリビュー](./query-view) を参照してください。

## 注意事項

- 同じノート内に複数の埋め込みビューがある場合、それぞれが独自の状態を持ちます。
- 埋め込みビューは外部変更通知を受け取ります — Todo ファイルが外部で変更された場合、自動的に再描画されます（300 ms デバウンス）。
- 空のコードブロック → フルインタラクティブパネル
- テキストありコードブロック → クエリ結果（読み取り + チェックボックストグルのみ）
