# ドラッグ＆ドロップ

ZenTodo はタスク行とリストタブの両方でドラッグ＆ドロップをサポートしています。

## タスクの並び替え

1. タスク行にホバーすると左端にドラッグハンドル（⠿）が表示されます。
2. ハンドルをクリックしてドラッグ。
3. ドロップインジケーター（線）が挿入位置を示します。
4. リリースしてドロップ。

### 制約

- タスクは **同じセクション内のみ** 並び替えできます（未完了は未完了の中、完了は完了の中）。
- セクションをまたぐ移動はできません。

### キャンセル

ドラッグ中に **Escape** を押すと並び替えなしでキャンセルできます。

## リストタブの並び替え

1. 任意のリストタブをクリックしてドラッグ。
2. 縦のインジケーター線が挿入位置を示します。
3. リリースしてドロップ。順序は自動的に保存されます。

### キャンセル

タブドラッグ中に **Escape** を押すとキャンセルできます。
