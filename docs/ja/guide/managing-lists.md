# リスト管理

## リストを作成する

タブバーの **+** をクリック。名前を入力して **Enter**。リストは設定した Todo フォルダに `.md` ファイルとして保存されます。

## リストを切り替える

タブバーの任意のタブをクリックして切り替えます。

## 「All」タブ

最初のタブには **全リストのタスクをまとめて** 表示します。完了タスクはこのビューでは非表示にして、未完了タスクに集中できるようにしています。

## タブを並び替える

タブをドラッグして並び替えられます。順序は自動的に保存され、Obsidian を再起動しても維持されます。

## リストを削除する

対応する `.md` ファイルを Vault から削除するとリストが消えます。ZenTodo が変更を検出してタブを自動的に削除します。
