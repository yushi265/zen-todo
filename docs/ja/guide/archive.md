# タスクのアーカイブ

## アーカイブとは

アーカイブは完了タスクをリストファイルの末尾にある `## Archived` セクションに移動する機能です。アーカイブされたタスクはファイルに残りますが、アクティブなタスクビューからは消えます。

## アーカイブする方法

**完了済みのルートレベルタスク** にホバーして **🗃️**（アーカイブ）アイコンをクリック。

タスクは `.md` ファイルの末尾の `## Archived` セクションに移動します：

```markdown
## Archived

- [x] 古い完了タスク ✅ 2026-01-15
```

## 注意事項

- アーカイブできるのは **完了済みのルートレベルタスク** のみです（サブタスクは直接アーカイブできません）。
- `## Archived` セクションは毎回の保存時にそのまま保持されます — ZenTodo がその内容を変更することはありません。
- アーカイブされたタスクはアクティブなタスクビューやクエリ結果には表示されません。
