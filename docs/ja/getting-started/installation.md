# インストール

## 手動インストール（コミュニティプラグイン掲載前）

1. [最新リリース](https://github.com/yushi265/zen-todo/releases) から `main.js`、`manifest.json`、`styles.css` をダウンロード。
2. Vault の `.obsidian/plugins/zen-todo/` にコピー。
3. Obsidian を再起動し、**設定 → コミュニティプラグイン** で **ZenTodo** を有効化。

## BRAT（ベータ版自動アップデートツール）

公式掲載前に自動アップデートを受け取りたい場合：

1. [BRAT プラグイン](https://github.com/TfTHacker/obsidian42-brat) をインストール。
2. **BRAT: Add a beta plugin** を実行し、`yushi265/zen-todo` を入力。
3. BRAT が自動でプラグインをインストール・更新します。
