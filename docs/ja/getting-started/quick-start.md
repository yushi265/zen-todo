# クイックスタート

## ZenTodo を開く

インストール後、以下のいずれかの方法で開けます：

- リボン（左サイドバー）の **チェックボックスアイコン**（☑）をクリック
- コマンドパレット: `ZenTodo: Open todo list`
- コマンドパレット: `ZenTodo: Create new todo list`

## 最初のリストを作成する

1. タブバーの **+** をクリックして新しいリストを作成。
2. リスト名を入力して **Enter** を押す。
3. リストは設定した Todo フォルダ（デフォルト: `30_ToDos`）に `.md` ファイルとして保存されます。

## 最初のタスクを追加する

1. パネル上部の入力フィールドをクリック。
2. タスクのテキストを入力。
3. 必要に応じてカレンダーアイコンをクリックして期日を設定。
4. **Enter** を押すか **追加** をクリック。
